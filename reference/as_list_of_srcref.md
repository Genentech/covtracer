# Create an S3 list of srcref objects

Create an S3 list of srcref objects

## Usage

``` r
as_list_of_srcref(x)

# S3 method for class 'environment'
as_list_of_srcref(x)

# S3 method for class 'list'
as_list_of_srcref(x)
```

## Arguments

- x:

  A list or single srcref to coerce to a `list_of_srcref`

## Value

A `list_of_srcref` class object
