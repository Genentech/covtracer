# Format a list_of_srcref object

Format list_of_srcref as character

## Usage

``` r
# S3 method for class 'list_of_srcref'
format(x, ..., full.names = FALSE, full.num = FALSE)
```

## Arguments

- x:

  A `list_of_srcref` object

- ...:

  Additional arguments unused

- full.names:

  A `logical` value indicating whether to use full file paths when
  formatting `srcref`s.

- full.num:

  A `logical` value indicating whether to use all numeric `srcref`
  components when formatting `srcref`s.

## Value

A `character` vector of formatted strings
